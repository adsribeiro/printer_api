# Printer API

API desenvolvida com FastAPI para gerenciar impressões em ambiente Windows, suportando tanto impressoras de etiquetas (Zebra/ZPL) quanto impressoras comuns (via desenho de texto/PDF).

## 🚀 Como começar
Veja o arquivo [PASSO_A_PASSO.md](./PASSO_A_PASSO.md) para instruções detalhadas de criação e execução.

## 🛠️ Tecnologias
*   Python 3.12+
*   FastAPI
*   pywin32 (Interação com API do Windows)
*   uv (Gerenciador de pacotes)
